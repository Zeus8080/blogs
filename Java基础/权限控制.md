#利用BigInteger实现权限控制


##1.实现方式


```
BigInteger bi = new BigInteger("0");
bi = bi.setBit(2);
bi = bi.setBit(4);

System.out.println("bi的值："+bi);
System.out.println("2是否包含在bi中："+bi.testBit(2));
System.out.println("4是否包含在bi中："+bi.testBit(4));
System.out.println("6是否包含在bi中："+bi.testBit(6));
```
结果是：
```
bi的值：20
2是否包含在bi中：true
4是否包含在bi中：true
6是否包含在bi中：false
```
